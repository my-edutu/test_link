# LinguaLink

## Overview
LinguaLink is a cross-platform mobile application built with React Native (Expo) designed to facilitate language learning and voice data collection, specifically optimized for the Nigerian market. The app combines a structured voice recording system with social features and gamification to create an engaging user experience (Voice Data Collection & Social Platform).

## Key Features
- **Voice Recording Platform**: High-quality audio recording with real-time waveform visualization for over 150+ curated prompts across multiple languages.
- **Social Networking**: Connect with other users through profiles, activity feeds, comments, likes, and direct messaging.
- **Gamification**: Earn points, unlock badges, and compete on leaderboards to stay motivated.
- **Offline-First**: Robust offline capabilities ensuring functionality even with intermittent connectivity.
- **Cross-Platform**: Native performant experience on both iOS and Android.

## Tech Stack
- **Frontend**: React Native, Expo, React Navigation
- **Backend/Auth/Database**: Supabase
- **Media**: Expo AV, Expo Camera, Expo FileSystem
- **Real-time**: Socket.io / WebRTC capabilities

## Getting Started

### Prerequisites
- Node.js
- npm/yarn

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
Start the Expo development server:
```bash
npx expo start
```
- Press `a` for Android emulator
- Press `i` for iOS simulator
- Scan the QR code with Expo Go on your physical device

## Project Structure
- `src/`: Core source code
- `assets/`: Static assets (images, fonts, etc.)
- `app.json`: Expo configuration
- `PROJECT.md`: Detailed project specifications and roadmap
